## Features

This project is a web application for browsing and booking IT events. Key features explored in this learning project include:

- **Event Browsing**: View a list of upcoming IT events with details and images.
- **Event Details**: Detailed pages for individual events utilizing dynamic routing (`[slug]`).
- **Event Booking**: Functionality to simulate booking a spot for an event.
- **Next.js App Router**: Utilizing the modern App Router architecture, Server Components, and specialized routing configurations.
- **API Routes**: Custom Next.js API routes (`app/api/...`) used to handle data fetching and booking submissions.
- **PostHog Integration**: Examples of integrating third-party analytics (PostHog) within a Next.js application.
- **Tailwind CSS**: Styling and responsive design using Tailwind CSS.

## Getting Started

To run this project locally:

1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Set up your `.env.local` file with necessary environment variables (e.g., PostHog keys, database URLs if applicable).
4. Run the development server:

   ```bash
   npm run dev
   ```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
